CMDH - command processor running in a hidden window.

Have you  ever wanted to be able to run a BAT file or some commands in CMD
window directly without watching the ugly black CMD window splashing on your
screen? If you have - this small program that will launch CMD window in hidden
mode and execute everything you want in it.

-------------------------------------------------------------------------------
Install:

Mouse Hider does not require any installation - unzip executable into any
appropriate folder and simply run it when you need.

-------------------------------------------------------------------------------
Uninstall:

Remove program from the folder you put it earlier.

-------------------------------------------------------------------------------
Supported OS:

2. Windows XP
2. Windows Server 2003
3. Windows 7
